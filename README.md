# Traffic Accident Analysis

This project was developed as part of my internship at Prodigy InfoTech to analyze traffic accident data and identify patterns related to road conditions, weather conditions, time of day, and accident hotspots.

Using Python, Pandas, Matplotlib, Seaborn, and Folium, I performed Exploratory Data Analysis (EDA), correlation analysis, and geospatial hotspot mapping to uncover key factors contributing to traffic accidents.

## Key Insights
- Wet roads recorded the highest number of accidents.
- Rainy weather significantly increased accident frequency.
- Evening and night hours showed higher accident occurrence.
- High-traffic locations were identified as accident hotspots.
- Environmental conditions influenced accident severity.

## Technologies Used

- Pandas
- Matplotlib
- Seaborn
- Folium
- Jupyter Notebook
